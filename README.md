# typescript-express-angular-module

##Get Started

#Install Dependencies and typings 
`npm install`

#build both client
`gulp build`

#test locally 
Run following from base directory 
`electron build`


#Build Installer with NSIS
`gulp build-electron` 
 You can check the installer created in 'dist'
